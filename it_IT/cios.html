    cIOS

    Per supporto in Inglese, chiedere aiuto sul server Nintendo Homebrew su Discord.

    Table of Contents

    • Requisiti
    • Istruzioni
      • Sezione I - Download
      • Sezione II - Installazione

    Nonostante i cIOS siano stati per la maggior parte soppiantati da AHBPROT, che dà l’accesso completo all’hardware, essi hanno comunque delle applicazioni utili. Per esempio, questo permette la funzionalità di USB loader come USB Loader GX e WiiFlow, insieme ad altre applicazioni homebrew come SaveGame Manager GX. Puoi saltare questo processo se vuoi, ma normalmente migliore il tuo Wii con quasi nessun lato negativo.

    Se hai un Wii mini, usa invece questa guida per cIOS. Provare a installare altri cIOS su un Wii mini non funzionerà.

    Requisiti

    • Un Wii
    • Una scheda SD o dispositivo USB
    • d2x cIOS Installer

    Assicurati che, se stai usando la scheda SD, l’interruttore della proteziona scrittura sia in posizione disattivata, altrimenti non potrai selezionare le opzioni corrette nell’installer.

    Istruzioni

    Sezione I - Download

    Se il tuo Wii ha una connessione a internet, puoi saltare questa sezione.
    Tuttavia, se incontri errori come net_init failed, net_gethostbyname failed:, o altri problemi durante la fase di download, questo permetterà al tuo Wii di saltare alla fase successiva.

    Se non sei su Windows, puoi scaricare & eseguire questo script, e scaricherà i file WAD per te.

    1. Spegni la console.
    2. Inserisci la scheda SD o dispositivo USB nel tuo computer.
    3. Sul tuo computer, scarica NUS Downloader.
    4. Copia tutti i file dal .zip di NUS Downloader a una cartella sul tuo computer.
    5. Esegui NUS Downloader.exe.
    6. Naviga a Database > IOS > IOS38, poi seleziona v4123.
      • Assicurati che “Pack WAD” sia attivato.
      • Non attivare “Patch IOS”. Quello è il ruolo del cIOS Installer.
    7. Clicca Start NUS Download!.
    8. Ripeti i passaggi sopra per IOS56 v5661, IOS57 v5918 e IOS58 v6175.
    9. Quando il processo è completato, ci sarà una cartella chiamata titles nella stessa cartella del NUS Downloader.
    10. Apri la cartella titles e navigaci finchè non trovi i quattro file WAD che hai scaricato.
    11. Piazza ogni file WAD sul root della tua SD o dispositivo USB.
      • Questo deve essere lo stesso dispositivo che contiene d2x cIOS Installer.

    Sezione II - Installazione

    1. Copia la cartella apps dal .zip di d2x-cios-installer al root della tua scheda SD o dispositivo USB.
    2. Reinserisci la tua scheda SD o dispositivo USB nella tua console.
    3. Accendi la console.
    4. Avvia l’Homebrew Channel.
    5. Avvia d2x cIOS Installer dalla lista degli homebrew.

    6. Set everything to match the following:

       Select cIOS <d2x-v11-beta1>
 Select cIOS base <38>
 Select cIOS slot <248>
 Select cIOS revision <65535>

    7. Press A. This will bring you to the slot map:

    8. Press A again.

    9. After the installation has finished, you will be brought back to the slot map. Ensure that the slot that was just installed is highlighted green.

      If the slot is highlighted red, try install the cIOS again.
      If you are getting an error before/during the downloading stages, press B to exit and continue from Section I.

    10. Press A. This will return you to the configuration screen.

    11. Repeat the previous 5 steps with the following configurations:

      • Installazione del cIOS 249

          Select cIOS <d2x-v11-beta1>
  Select cIOS base <56>
  Select cIOS slot <249>
  Select cIOS revision <65535>

      • Installazione del cIOS 250

          Select cIOS <d2x-v11-beta1>
  Select cIOS base <57>
  Select cIOS slot <250>
  Select cIOS revision <65535>

      • Installazione del cIOS 251

          Select cIOS <d2x-v11-beta1>
  Select cIOS base <58>
  Select cIOS slot <251>
  Select cIOS revision <65535>

    12. Once you have finished with all 4 cIOSes, press B to return to the Homebrew Channel.

    Continue to Open Shop Channel Installation
    Now that your Wii has adequate brick protection, you can install the Open Shop Channel, a trusted repository for homebrew that can be accessed both on and off the Wii.
