    Priiloader

    Priiloader è un’utilità essenziale creato da DacoTaco che aggiunge un livello di protezione da brick sul tuo Wii. Si carica prime se si carichi il Menu di Sistema Wii. Può anche applicare hack che possono essere utilizzate per migliorare, sbloccare e/o correggere molte funzioni del Menu di sistema.

    Inoltre, può essere usato per avviare rapidamente titoli e homebrew, oppure per cambiare l’ordine di autoboot del Wii.

    Per ulteriori informazioni su Priiloader e sulle sue opzioni, puoi vedere la pagina Usare Priiloader.

    Requisiti

    • Una scheda SD o dispositivo USB
    • Priiloader Installer (il file .zip)
    • Load Priiloader (il file .zip)

    Requisiti facoltativi per vWii (altamente raccomandati)

    • Una scheda SD
    • Priiloader Wii U Forwarder (download diretto)
      • Aroma deve essere installato sulla console affinché il Priiloader Wii U Forwarder possa funzionare.

    Istruzioni

    Se stai usando un vWii:

    • NON installare un tema dopo aver installato Priiloader, altrimenti c’è il rischio di BRICK sul vWii!
    • Priiloader non avrà effetto su titoli VC per Wii U.

    Sezione I - Preparazione

    1. Spegni la console.
    2. Inserisci la scheda SD o dispositivo USB nel tuo computer.
    3. Copia la cartella apps dal .zip di LoadPriiloader al root della tua scheda SD o unità USB.
    4. Copia la cartella apps dal .zip di Priiloader al root della tua scheda SD o unità USB.
    5. (Solo per vWii) Copia la cartella wiiu dal .zip di PriiloaderWiiUForwarder al root della scheda SD.
    6. Reinserisci la tua scheda SD o dispositivo USB nella tua console.

    Sezione II - Installazione di Priiloader

    1. Accendi la console.
    2. Avvia l’Homebrew Channel.
    3. Avvia Priiloader Installer dalla lista degli homebrew.

    4. Premi il pulsante + sul telecomando Wii oppure il pulsante A su un controller GameCube.

    5. Premi A per tornare all’Homebrew Channel.

    Sezione III - Avvio di Priiloader

    1. Premi il pulsante HOME e seleziona Ritorna al Menu di Sistema per uscire dall’Homebrew Channel.

    2. La tua console quindi caricherà il menu di Priiloader.

    Successivamente, per entrarci dovrai:

    • Tenere premuto il pulsante RESET sul Wii durante l’accensione
      • Solo Wii
    • Tenere premuto il tasto ESC su una tastiera USB connessa durante l’accensione della console
      • Wii, vWii e mini
    • Eseguire “Load Priiloader” dall’HBC
      • Wii, vWii e mini
    • Usare il “Priiloader Wii U Forwarder”
      • Solo vWii

    Sezione III - Configurazione di Priiloader

    1. Scorri in basso fino a System Menu Hacks e premi A.

      Se hai inserito il Priiloader Installer sulla tua unità USB, assicurati di non avere una scheda SD inserita allo stesso tempo.
      Altrimenti, Priiloader potrebbe fallire a trovare il file hacks_hash.ini.

    2. Scorri fino a ciascuna delle seguenti opzioni e premi A:
      • Block Online Updates
      • Se NON stai usando un display CRT: Remove Deflicker
    3. Per le seguenti opzioni, attiva le opzioni in base alla tua console:
      • Wii:
        • Se stai usando la modalità 480p: 480p graphics fix in the system menu
        • Block Disc Updates
      • Wii mini:
        • Blocca Aggiornamenti da Disco
    4. Scorri in basso fino a save settings e premi A.
    5. Premi B per tornare al menu principale.
